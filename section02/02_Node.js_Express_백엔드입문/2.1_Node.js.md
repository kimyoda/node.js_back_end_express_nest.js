# 2.1 Node.js

Node.js는 서버 측 자바스크립트 런타임 환경이다.
2009년 오픈소스로 공개했다.
Node.js는 자바스크립트를 사용하는 V8 엔진을 사용한다.
Node.js이전에 논블로킹/비동기 API를 서버 환경에 구현이 힘듬 -> 스레드와 공유 자원을 직접 관리가 어려움
위와 같은 개발환 경을 논블로킹 감옥이라 표현.

> 논블로킹은 함수 실행 완료를 기다리지 않고 다음 코드를 실행하는 것

이벤트 기반(Event-Driven) 비동기 환경, Node.js 탄생

2010년에 npm이라는 패키지 매니저 공개.
이후 JS재단과 Node.js 재단이 오픈JS(OpenJS)재단으로 합쳐짐
