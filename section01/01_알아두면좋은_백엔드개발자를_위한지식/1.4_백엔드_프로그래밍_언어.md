# 1.4 백엔드 프로그래밍 언어

## 🐍 Python (파이썬)

데이터 분석 분야에서 파이썬은 서버 개발에도 널리 사용한다.
문법이 간결하고 생산성이 높아 AI/ML 서비스에서 많이 채택된다.

프레임워크
**- Flask**: 경량 마이크로 프레임워크. 소규모 API 서버에 적합하다.
**- Django**: 풀스택 웹 프레임워크. ORM. 인증, 관리자 페이지 등 기능이 내장되어 있다.
**- FastAPI**: 비동기 처리와 자동 문서화(Swagger)를 지원하는 현대적인 프레임워크.

## ☕ Java (자바)

스프링(Spring)프레임워크를 기반, 대규모 서비스에서 오랫동안 사용되어있다.
안정성과 생태계가 매우 성숙, 금융, 공공 분야에서 강세다

주요 프레임워크

- Spring Boot: 설정을 최소하화고 빠르게 서버를 구성할 수 있는 자바 표준 프레임워크다.

## 🟦 JavaScript / TypeScript

원래 프론트엔드 언어였지만, Node.js의 등장으로 백엔드에서도 폭넓게 사용된다. 프론트엔드와 같은 언어를 쓰기 때문에 풀스택 개발에 유리하다.
**- Express**: 가볍고 유연한 Node.js 웹 프레임워크. 자유도가 높다
**- NestJS**: TypeScript 기반의 구조화된 프레임워크다. 대규모 서비스에 적합하다.

> 타입스크립트
> 자바스크립트의 슈퍼셋인 오픈소스 프로그래밍 언어이다.
> MS에서 개발, 유지보수하고 있고 엄격한 문법을 지원한다.
> 타입스크립트에서 자신이 원하는 타입을 정의하고 프로그래밍을 하며 자바스크립트로 컴파일해 실행할 수 있다.

## 🔵 C#

Microsofot 생태계와 밀접, 국내보다 미국에서 더 많이 사용된다. 게임 서버 개발(.NET기반)과 서비스에 강점이 있다.
주요 프레임워크

- **ASP.NET Core**: 성능 크로스플랫폼 웹 프레임워크, API서버와 MVC 패턴을 모두 지원한다.

## ⚙️ C++

성능이 최우선인 환경에서 사용된다. 메모리를 직접 제어할 수 있어 게임 서버, 금융 시스템 등 관련되고 개발 난이도와 유지보수 비용이 높다.

## 🐹 Go (Golang)

Google이 개발한 언어, 단순한 문법과 뛰어난 동시성 처리 능력이 특징이다. 컴파일 속도가 빠르고 바이너리 하나로 배폭 ㅏ간편해 Docker, Kubernetes도 Go로 작성되었따.

주요 프레임워크
**- Gin**: 가볍고 빠른 HTTP 웹 프레임워크
**- Echo**: 고성능 미니멀 프레임워크
**- Fiber**: Express에서 영감을 받은 프레임워크, Node.js 개발자에게 친숙

## 🟣 Kotlin (코틀린)

JetBrains에서 개발한 언어 JVM 위에 동작하며 자바와 100%호환. 자바보다 간결하고 안전한 문법을 제공, 안드로이드 공식 언어.
서버 개발에서 자바를 대체하는 흐름이 있다.

주요 프레임워크
**- Ktor**: Kotlin 전용으로 설계된 비동기 웹 프레임워크
**- Spring Boot(Kotlin)**: 기존 Spring Boot를 코틀린으로 그대로 사용할 수 있어 자바 활용능력이 높다면 사용하기 편하다

## 🦀 Rust (러스트)

Mozilla가 개발, C/C++ 수준의 성능을 내며 메모리 안전성을 언어 차원에서 보장한다.
소유권(Ownership) 시스템으로 런타임 에러, 메모리 누수를 컴파일 단계에서 차단한다.

주요 프레임 워크
**- Actiox-web**: 벤치마크 성능을 자랑하는 웹 프레임워크
**- Axum**: 비동기 프레임워크로 빠르게 적용중.
